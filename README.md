# RS app Back-end


## To deploy application

 1. `mvn clean install`     deploy application
 2. `cd cdk `               go to cdk deployment module
 3. `cdk deploy`            deploy application

## To deploy synth

1. `mvn clean install`     deploy application
2. `cd cdk `               go to cdk deployment module
3. `cdk synth`             synth application
